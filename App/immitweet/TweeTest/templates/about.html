How it works:

The app works by first downloading all of someone's tweets. This is done using both python-twitter and tweepy Python libraries. Next, the app applies a markov model to the person's tweets, structuring their tweets into one combined model. This model is fluid, meaning that if you put in the same handle twice, you will get different results. This tweet is then printed to the website through an HTTP Response, and posted to a remote twitter (@internetofroas1).
